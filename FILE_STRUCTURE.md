## ğŸ“ File Naming Conventions

### Components
- **File**: `ComponentName.jsx`
- **Function**: `export default function ComponentName() {}`
- **Location**: `src/components/ComponentName.jsx`

### Pages
- **File**: `PageName.jsx`
- **Function**: `export default function PageName() {}`
- **Location**: `src/pages/PageName.jsx`

### Styles
- **Global**: `src/index.css`
- **Component-specific**: Keep in component file or `src/styles/ComponentName.css`

### Constants/Config
- **Location**: `src/constants/` or top of component
- **Naming**: `CONSTANT_NAME` for constants

## ğŸ”„ File Modification Guide

### To Add a New Page
1. Create `src/pages/NewPageName.jsx`
2. Export component with proper structure
3. Add route in `src/App.jsx`:
   ```jsx
   <Route path="/new-path" element={<NewPageName />} />
   ```
4. Add link in `src/components/Header.jsx`

### To Update Styling
1. Edit `src/index.css` for global styles
2. Edit `tailwind.config.js` for theme
3. Use Tailwind classes in components

### To Add Dependencies
1. Run: `npm install package-name`
2. For dev: `npm install -D package-name`
3. Import in your components

### To Update Routes
1. Edit `src/App.jsx`
2. Add/remove `<Route>` elements
3. Ensure page components exist
